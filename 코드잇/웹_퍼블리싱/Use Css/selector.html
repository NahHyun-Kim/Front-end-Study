.nthchild p:nth-child(n) {} : n번째 p태그에만 속성 적용
.nthchild p:first-child {} : .nthchild의 자식인 p 태그 중 첫 번째에만 속성 적용
div1 클래스 밑의 모든 태그에 적용하고 싶다면 .div1 > * 로 적용

마우스 오버(hover) : 마우스를 갖다댔을때 스타일 속성이 바뀌도록 설정
h1 {} 스타일 지정 후, h1:hover {} 를 통해 지정하면 마우스를 갖다댈 시 hover 에 지정된 속성으로 스타일 적용

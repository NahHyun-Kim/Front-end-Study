div class="div1"
안에 p태그, h1 태그를 입력 후
css로 .div1 { color : red; } 를 입력할 경우 p태그와 h1태그에도 스타일이 적용된다.

상속되는 속성들

color, font-family, font-size, font-weight,
line-weight, line-height, list-style, text-align, visibility...

a태그의 경우에는 상속이 되지 않는데, color: inherit; 을 통해 강제로 상속시킬 수 있다.

CSS 우선 순위

h1에 대한 color 속성을 두번 지정했다면, 이전에 나온 스타일을 덮어쓴다.

명시도 계산기 - inline, id수, 선택자의 class-attribute-pseudo-class, 일반 요소 순대로 명시도를 주어, 높은 곳의 스타일을 적용
